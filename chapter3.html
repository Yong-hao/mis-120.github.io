Chapter 3: Number System


Number systems are the technique to represent numbers in the computer system architecture, every value that you are saving or getting into/from computer memory has a defined number system. Computer architecture supports following number systems:

  • Binary (Base 2) — Represent any number using 2 digits [0–1]
  • Decimal (Base 10) — Represent any number using 10 digits [0–9]
  • Octal (Bas3 8) — Represent any number using 8 digits [0–7]
  • Hexadecimal (Base 16) — Represent any number using 10 digits and 6 characters [0–9, A, B, C, D, E, F]

A number system is defined as a system of writing to express numbers. A number can be written differently in different bases.

  • Cave dweller: IIIII
  • Roman: V
  • Arabic: 5
